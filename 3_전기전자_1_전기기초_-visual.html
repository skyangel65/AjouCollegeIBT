계산 문제 완전정복 공식표
오옴의 법칙: E = IxR  |  I = E/R  |  R = E/I 전력: P = ExI = I²xR = E²/R 직렬 합성: R합 = R1 + R2 + R3 ... 병렬 합성: 1/R합 = 1/R1 + 1/R2 + 1/R3 ...
기출 계산 패턴 TOP 5
유형①
20Ω 저항에 2A → 전압은?
40V
E = IxR = 2x20 = 40V
유형②
2Ω, 3Ω, 6Ω 병렬 합성저항은?
1Ω
1/R = 1/2+1/3+1/6 = 3/6+2/6+1/6 = 6/6 = 1 → R=1Ω
유형③
12V, 직렬 2Ω+4Ω+6Ω → 전류는?
1A
R합=12Ω, I=12÷12=1A
유형④
12V에 60W + 30W 전구 병렬 → 전체 전류는?
7.5A
P합=90W, I=P/E=90÷12=7.5A
유형⑤
3,300V → 110V 변압기 권선비는?
30:1
3300÷110=30 → 1차:2차 = 30:1

⚠️ 시험에서 자주 틀리는 함정 유형 TOP 10
함정①
부특성 서미스터는 온도가 높아지면 저항이 커진다?
❌ 틀림. 부특성(NTC) = 온도↑ → 저항↓
온도↑→저항↑는 정특성(PTC). 자동차에서는 NTC 사용.
함정②
발광 다이오드는 PN 접합면에 역방향 전압을 걸면 빛이 난다?
❌ 틀림. LED는 순방향 전압 인가 시 빛 발생. 역방향 도통은 제너 다이오드.
함정③
사이리스터 단자는 애노드, 캐소드, 컬렉터?
❌ 틀림. SCR = 애노드(A), 캐소드(K), 게이트(G). 컬렉터는 트랜지스터 단자.
함정④
콘덴서 정전용량은 절연물의 절연도에 반비례한다?
❌ 틀림. 절연도에는 비례. 반비례하는 것은 오직 극판 사이의 거리뿐.
함정⑤
주파수는 직류의 파형이 반복되는 비율이다?
❌ 틀림. 주파수는 교류의 파형이 반복되는 횟수(Hz). 주기의 역수.
함정⑥
트랜지스터의 릴레이와 같은 작용은 정류 작용이다?
❌ 틀림. 릴레이와 같은 ON/OFF는 스위칭 작용. 정류는 다이오드 기능.
함정⑦
파워 트랜지스터 컬렉터 단자는 점화 2차코일과 연결?
❌ 틀림. 컬렉터는 점화 1차코일 (-) 단자에 연결. 이미터는 접지.
함정⑧
TPMS 타이어 압력센서 배터리 수명은 영구적이다?
❌ 틀림. TPMS 센서 배터리는 소모품. 수명이 다하면 센서 자체 교환 필요.
함정⑨
고장코드는 점화스위치 OFF 시 자동 삭제된다?
❌ 틀림. 고장코드는 ECU에 배터리 전원으로 유지됨. 배터리 (-)단자 분리해야 삭제.
함정⑩
AND 게이트: 입력 A=1, B=1, C=1, D=0 → 출력 1?
❌ 틀림. AND 회로는 입력 모두 1일 때만 출력 1. D=0이 있으므로 출력 0.

Q16 [기출]
