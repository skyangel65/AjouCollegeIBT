세부 출제 패턴 (반드시 알아야 할 것)
주제출제 횟수핵심 포인트
오옴의 법칙 계산5문E=I×R, 직렬/병렬 합성 후 전류 구하기
전력(W) 계산4문P=EI, I=P/E 패턴 반복
트랜지스터3문스위칭 작용=릴레이, NPN형 자동차 사용
다이오드 종류3문LED(순→빛), 포토(빛→전류), 제너(정전압)
서미스터·SCR3문NTC=온도↑저항↓, SCR단자=A·K·G
콘덴서 정전용량2문극판거리에 반비례 (나머지 모두 비례)
변압기 권선비1문V1/V2 = N1/N2
논리회로 기호1문NOR = OR+반전(○ 붙음)
계산 풀이 5단계 공식 12문 커버
Step 1. 직렬: R합 = R₁ + R₂ + …
Step 2. 병렬: 1/R합 = 1/R₁ + 1/R₂ + …
Step 3. 직병렬: 병렬 먼저 → 직렬과 합산
Step 4. 전류: I = E / R합
Step 5. 전력: P = E × I   또는   I = P / E
기출 계산 6가지 패턴 암기
패턴① 직렬 전류
R = 2+4+6 = 12Ω
I = 12V ÷ 12Ω = 1A
패턴② 직·병렬 전류
병렬 2Ω‖2Ω = 1Ω
R합 = 1+1 = 2Ω → I = 5A
패턴③ 전압강하
I = E/R합 = 2A
E₂Ω = I×R = 2×2 = 4V
패턴④ 전력→전류
P합 = 60+30 = 90W
I = P/E = 90÷12 = 7.5A
패턴⑤ 병렬 합성저항
1/R = 1/1+1/3+1/5 = 23/15
R = 15/23 Ω
패턴⑥ 변압기 권선비
3300 ÷ 110 = 30:1
V₁/V₂ = N₁/N₂
💡 헤드라이트 2개 병렬: 전체 전류에서 1개 출력 = E × (I/2)
실수 방지 체크리스트
병렬 합성저항은 가장 작은 저항보다 더 작다 → 계산 후 검증 필수
직렬은 같은 전류, 병렬은 같은 전압 → 무엇을 구하는지 먼저 확인
P=EI에서 I=P/E (분모가 E) — 헷갈리면 단위로 확인: W÷V=A
헤드라이트 회로에서 10A는 양쪽 합산 → 1개당 출력 = E×(I÷2)
다이오드 4종 완전 비교 6문 커버
종류동작 원리핵심 키워드자동차 용도
실리콘 다이오드순방향만 통전정류충전회로
제너 다이오드역방향 일정전압 이상 → 도통 유지정전압 회로교류발전기 전압조정기
LED (발광)순방향 전류 → 빛 발생전류→빛파일럿램프, 크랭크각센서
포토 다이오드빛 → 역방향 전류 흐름빛→전류라이트센서, 차고센서, TDC
❌ 틀린 선지: "발광 다이오드는 역방향 전압을 걸면 빛이 난다" → 순방향이어야 함
❌ 틀린 선지: "빛을 받으면 전압이 생긴다"는 LED 설명 → 포토 다이오드 혼동 주의
트랜지스터 핵심 정리 3문 커버
3단자: 이미터(E), 베이스(B), 컬렉터(C)
스위칭 작용 = 릴레이와 같은 ON/OFF
증폭 작용 = 베이스로 큰 전류 제어
자동차 → NPN형 사용
포토 트랜지스터
빛으로 컬렉터 전류 제어
조향각센서, 차고센서 사용
CDS라고도 함
서미스터 & 사이리스터 3문 커버
서미스터 (Thermistor)
NTC(부특성): 온도↑ → 저항↓
PTC(정특성): 온도↑ → 저항↑
일반적으로 NTC 사용
냉각수온센서, 흡기온도센서
출제: "부특성=온도↑→저항↑" → 틀림!
사이리스터 (SCR)
단자: 애노드(A) · 캐소드(K) · 게이트(G)
PNPN 4층 구조
스위칭 작용만 함
도통 후 게이트 제거해도 계속 도통
출제: "컬렉터가 SCR 단자" → 틀림!
광도전 셀 (CdS) 오토라이트 조도센서
빛 조사량 증가 → 저항 감소 / 빛 감소 → 저항 증가
가시광선 감도 높음 → 가로등 자동점멸, 카메라 노출계, 오토라이트
⚠️ 시험에서 자주 틀리는 함정 유형 TOP 8
함정①
부특성 서미스터는 온도가 높아지면 저항이 커진다?
❌ 틀림. 부특성(NTC) = 온도↑ → 저항↓
온도↑→저항↑는 정특성(PTC). 자동차에서는 NTC 사용.
함정②
발광 다이오드는 PN 접합면에 역방향 전압을 걸면 빛이 난다?
❌ 틀림. LED는 순방향 전압 인가 시 빛 발생. 역방향 도통은 제너 다이오드.
함정③
사이리스터 단자는 애노드, 캐소드, 컬렉터?
❌ 틀림. SCR = 애노드(A), 캐소드(K), 게이트(G). 컬렉터는 트랜지스터 단자.
함정④
콘덴서 정전용량은 절연물의 절연도에 반비례한다?
❌ 틀림. 절연도에는 비례. 반비례하는 것은 오직 극판 사이의 거리뿐.
함정⑤
자성체란 전자유도에 의해 자화되는 물질이다?
❌ 틀림. 자성체는 자기유도에 의해 자화되는 물질.
함정⑥
주파수는 직류의 파형이 반복되는 비율이다?
❌ 틀림. 주파수는 교류의 파형이 반복되는 횟수(Hz). 주기의 역수.
함정⑦
트랜지스터의 릴레이와 같은 작용은 정류 작용이다?
❌ 틀림. 릴레이와 같은 ON/OFF는 스위칭 작용. 정류는 다이오드 기능.
함정⑧
SCR에서 캐소드→게이트 방향이 순방향이다?
❌ 틀림. SCR 순방향 = 애노드→캐소드, 게이트→캐소드 방향.

❌ 틀림 (이것이 정답 오답지)
NTC 부특성 = 온도↑→저항↓. 커지는 것은 PTC(정특성).
