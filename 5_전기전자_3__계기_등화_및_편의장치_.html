⚙️ 1. 계기류 — 속도계 / 수온계 / 유압계 / 연료계
계기방식핵심 원리
속도계 자석식 자석 회전 → 맴돌이 전류 발생 → 스프링 장력과 평형 → 지침 이동 ★★★
디지털식 차속센서 → 전기신호 변환 → 컴퓨터 계산 → 숫자/그래프 표시
수온계 바이메탈식 서미스터(온도↑ → 저항↓ → 전류↑ → 바이메탈 변형↑ → H방향) ★★
유압계 인디케이터식 유압 낮음 → 다이어프램 수축 → 접점 닫힘 → 경고등 점등 ★★★
연료계 바이메탈식 연료 가득(F) → 플로트↑ → 저항↓ → 전류↑ → 바이메탈 변형↑
공통 — 밸런싱 코일식 2개 코일(L1, L2) 전류 → 자력 → 가동철편 당김 → 지침 이동
※ 계기판 수온계 미작동 점검 → CTS(Coolant Temperature Sensor) 확인 | 속도계 미작동 → VSS(Vehicle Speed Sensor) 확인 | 회전계 미작동 → CPS(Crankshaft Position Sensor) 확인
💡 2. 전조등(Head Light)
종류특징
실드 빔형 ★★★ 렌즈 + 반사경 + 필라멘트 일체형. 수명 길고 광도 안정. 단 하나라도 이상시 전체 교환
세미 실드 빔형 렌즈+반사경 일체, 전구만 분리 교환 가능. 먼지·습기 유입으로 광도 변화 가능
할로겐 전구 텅스텐 증발 → 할로겐 원소 결합 → 고온에서 해리 → 필라멘트 재부착 (재생순환반응, halogen cycle). 용량 60W/55W
HID 램프 제논(Xenon) 가스, 고압 20,000V 방전, 필라멘트 없음
공식조도[Lux] = 광도[cd] ÷ 거리²[m²]  →  25,000cd ÷ 100² = 2.5 Lux ★★
반사경알루미늄 반사율 90% / 은 92%(내구성 약) / 크롬 65%(내구성 좋음)
🔆 3. 방향지시등 / 미등 / 제동등
필수방향지시등 점멸: 1분에 60~120회. 현재 트랜지스터식(전자식) 사용
회로미등 회로: 차폭등 + 번호판등 + 계기판 조명등 → 병렬 연결
필수제동등 밝기: 미등의 3배 이상. 전구 단선 시 경고 기능 포함

🛡️ 5. 안전·편의 장치 핵심
★★와이퍼 전동기: 페라이트 자석식(브러시 3개, 저/고속 변환) / 복권식(저속=복권, 고속=직권). 스위치 OFF → 자동 정위치 정지
★★후진 경보(BWS): 1차(120cm↓ 340ms간격) / 2차(80cm↓ 170ms) / 3차(40cm↓ 연속). 거리 D = T×V/2
★TPMS 저압경고: 26~27psi↓ 점등, 30~31psi↑ 소등. 타이어 압력센서 무게 약 40g, 배터리 수명 10년
★에어백: 충돌 0.15초 → 가스발생기 작동 / 0.2초 → 팽창 시작 / 0.35~0.4초 → 완전 팽창 / 0.4~0.8초 → 운전자 충돌+가스 배출. SRS(안전벨트 보조장치)
★ETACS: 뒷유리 열선 15분 작동 / 파워윈도우 IG OFF 후 30초 유지 / 룸램프 소등 5~6초 지연
